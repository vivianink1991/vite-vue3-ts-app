## 技术栈
- JS框架：Vue3
- 语言：Typescript
- 路由：Vue Router 4
- 状态管理：Vue Store 4
- CSS预编译：SCSS
- 构建工具：Vite 2
- UI 框架：Element Plus
- HTTP 工具：Axios
- Mock 工具：Mockjs + vite-plugin-mock
- Git Hook 工具：husky + lint-staged
- 代码规范：EditorConfig + Prettier + ESLint + Airbnb JavaScript Style Guide
- 提交规范：Commitizen + Commitlint
